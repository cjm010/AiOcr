# Sponsor Evaluation License Outline

This document is a plain-English discussion draft only. It is not legal advice and is not a final contract.

## Purpose

The sponsor may review and evaluate the software for the sponsored academic project.

## What the sponsor is allowed to do

- access the code and related materials for review
- run the software internally for evaluation and testing
- discuss feedback with the student team
- use demo outputs for project evaluation

## What the sponsor is not allowed to do

- deploy the software into production
- use the software for commercial operations
- share the code with affiliates, vendors, or third parties
- modify the code for business use
- create derivative works for internal or external operational use
- remove notices of ownership or attribution
- claim ownership of the software unless a separate signed agreement says so

## Ownership

- ownership remains with the lawful rights holders
- this remains subject to any rights or policies of The University of Texas at Dallas
- this also remains subject to any sponsor agreement already in place

## Term

- the evaluation license should last only for a short defined period, such as 60 or 90 days
- when the evaluation period ends, the sponsor must stop using the software unless a commercial or transfer agreement is signed

## Confidentiality

- the sponsor should keep the code and related materials confidential
- the sponsor should not disclose the code, architecture, prompts, extraction logic, or learning artifacts to third parties

## No warranty

- the software is experimental and provided as-is
- the student team does not guarantee performance, accuracy, security, or fitness for business use

## Commercial use

- if the sponsor wants to use the software beyond evaluation, the sponsor must enter into a separate written agreement
- that later agreement should cover payment, scope of use, support, liability, maintenance, and intellectual property rights

## Learning data and outputs

- reviewed examples, learning artifacts, templates, and evaluation outputs should not be reused for commercial deployment without written approval
- if the sponsor wants continued use of these artifacts, that should be handled in the follow-on agreement
